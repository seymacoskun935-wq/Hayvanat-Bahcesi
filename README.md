# 🐾 Hayvanat Bahçesi Console Uygulaması (OOP Projesi)

Bu proje C# ile geliştirilmiş bir **console tabanlı hayvanat bahçesi simülasyonudur**.  
Amaç, OOP (Object Oriented Programming) kavramlarını öğrenmek ve uygulamaktır.

---

## 🚀 Özellikler

- 🐾 Hayvan ekleme (Kuş, Dinozor, Normal Hayvan)
- 📋 Hayvanları listeleme
- 🔊 Ses çıkarma (Polymorphism)
- 🍽️ Hayvan besleme sistemi
- ❌ Hayvan silme
- 🧹 Listeyi temizleme
- 🔍 Hayvan arama (case-insensitive)
- 🦖 Dinozorları listeleme
- 🕊️ Uçabilen hayvanları görüntüleme (Interface kullanımı)

---

## 🧠 Kullanılan OOP Konuları

- Class & Object
- Constructor
- Inheritance (Miras)
- Polymorphism (Override)
- Virtual / Override methods
- Interface (IUcabilenCanlilar)
- Type checking (`is` keyword)
- List yönetimi
